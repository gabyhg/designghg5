  • Name: The Positivitype

  • Category: Ecommerce and Virtual Reality Website

The Positivitype is an ecommerce website to sell typefaces, lettering and goods. It is linked to a virtual gallery where I feature my type and lettering work in a virtual reality environment. The goal is for users to have fun in a fully interactive and immersive experience.
